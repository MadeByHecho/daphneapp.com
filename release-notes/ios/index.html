Daphne

Release Notes

Version 2.7.0

Wednesday, February 16, 2017

[Improve] Support for location based reminders

[Add] Info dialog to promote new and existing features

[Improve] Date parsing of reminders like 'tonight at 7:30' and support for times like '730' or '1125'

[Improve] Support for recurring reminders like every 30 days

Version 2.6.0

Monday, January 2, 2017

[Add] Ability to set a default snooze option (in settings) that will be used when snoozing a reminder from a notification

[Add] Additional snooze options

[Fix] Issue when following a deep link to a reminder

Version 2.5.0

Monday, November 14, 2016

[Add] Support for location reminders (e.g. 'Take the trash out when I get home', 'Buy coffee when I get to the grocery store').

[Add] Support for parsing some time zones when trying to set reminders (e.g. 'Watch The Voice Tuesdays at 7 PM eastern').

[Add] More sample reminders to show off some of the newer features of Daphne's parsing abilities.

[Improve] Support for compound dates like 'Go to soccer practice every Tuesday and Thursday'

[Improve] Support for recurring reminders in Spanish

Version 2.4.1

Thursday, October 6, 2016

[Fix] Issue where daily repeating reminders wouldn't properly recur

[Fix] Issue where editing text after Daphne found a date could be frustratingly slow

Version 2.4.0

Tuesday, September 27, 2016

[Add] A Messages Extension for iOS 10 so you can send reminders to friends and family.

[Improve] iOS 10 Support

[Improve] Support for reminders with half increments like 3.5 hours

Version 2.3.0

Monday, September 12, 2016

[Add] A Share extension to quickly save reminders from other apps

[Improve] Date parsing of specific date reminders like 'December first'

[Fix] Pasting text that ended with a new line used to not be handled properly, NOW IT IS

[Add] Option to show all reminders (regardless of their immediacy) in the today extension

[Improve] iOS 10 Support so it can work great on your iPhone 7

Version 2.2.1

Wednesday, August 17, 2016

[Add] A 'Whats New' screen to let everyone know whats been added to Daphne

[Improve] Date parsing of bill paying type reminders like 'on the 15th of every month'

Version 2.2.0

Saturday, August 13, 2016

[Add] A new Pro plan that unlocks new features

[Add] The ability to sync reminders between your iOS devices (PRO only)

[Add] The ability to change the default reminder sound (PRO only)

[Add] Support for undoing delete/complete actions (just shake to undo).

[Add] Portuguese and French localizations

[Improve] Date parsing of Portuguese and French reminders

[Fix] Occasional crash when creating reminders

[Fix] An issue handling emoji in reminders
